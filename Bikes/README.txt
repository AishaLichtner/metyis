How to use code:

1. Install libaries from requirements.txt in terminal using:
	pip install -r	requirements.txt
2. Download datasets OD_****.csv and Stations_***.csv , stations.json for years that should be analysed and save in one folder
3. Start jupyter:
	jupyter notebook bikes.ipynb
4. Adjust path to folder where data was saved, years to be analysed and number of most frequent trips to be determinded (year_one, year_two, path, N)
5. Run all cells in notebook
